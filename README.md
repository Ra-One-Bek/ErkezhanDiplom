# 🌱 PlantCare AI — Приложение для распознавания растений и их заболеваний

PlantCare AI — мобильное приложение, разработанное с использованием Flutter для распознавания комнатных растений и заболеваний растений по изображениям.

Приложение использует модель машинного обучения TensorFlow Lite для классификации изображений, Firebase для авторизации пользователей и хранения истории, а также Gemini AI для генерации подробных рекомендаций по лечению и профилактике заболеваний.

---

# 📌 Описание проекта

Основная цель проекта — создание интеллектуального мобильного приложения, которое позволяет:

* распознавать растения по фотографии
* определять заболевания растений
* отображать точность предсказания модели
* сохранять историю распознаваний
* генерировать рекомендации через искусственный интеллект
* предоставлять подробную информацию о заболевании

Данный проект разработан в рамках дипломной работы.

# 🔥 Настройка Firebase

Проект использует:

* Firebase Authentication
* Cloud Firestore

Необходимо открыть Firebase Console и создать новый проект.

Включить авторизацию:

```text
Authentication
↓
Sign-in Method
↓
Email/Password
↓
Enable
```

Включить Firestore:

```text
Firestore Database
↓
Create Database
↓
Start in Test Mode
```

После этого выполнить:

```bash
flutterfire configure
```

Команда автоматически создаст:

```text

```

# ⚠ Важная информация

Приложение может распознавать только те растения и заболевания, которые присутствуют в обучающем датасете.

Если пользователь загрузит неизвестное растение, может появиться сообщение:

```text
Растение не распознано
```

Низкая уверенность модели обрабатывается автоматически.

# 🚨 Возможные проблемы

## TensorFlow Lite не работает в браузере

Пакет `tflite_flutter` использует:

```text
dart:ffi
```

Поэтому Web версия не поддерживается.

Запускать необходимо через Android:

```bash
flutter run
```

---

## Ошибка Firebase Authentication

Например:

```text
Chain validation failed
```

Причина:

Неправильно установлены дата и время устройства.

Решение:

Установить правильные дату и время.

---

## Gemini AI не работает

Проверить:

```text
frontend/.env
```

Убедиться, что там присутствует:

```env
GEMINI_API_KEY=ВАШ_API_КЛЮЧ
```

---

## Ошибка при загрузке в GitHub

Не загружать:

```text
build/
.dart_tool/
dataset/
dataset_raw/
venv/
.env
```

Оставить:

```text
frontend/assets/models/plant_disease_model.tflite
```
